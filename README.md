# General

now you can use &amp;&amp; in commands like in linux

The plugin only works for players who have
permission `commandsexploder.use`

# Examples

Player:
/say Hi!! &amp;&amp; me Hi everyone

Console:
say Hi!! &amp;&amp; me Hi everyone
